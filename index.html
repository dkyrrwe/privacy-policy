Privacy Policy

Last updated: March 8, 2026  ·  Nasiyachi App

01

Introduction

Nasiyachi is a mobile application designed for shop owners to manage customer debts and payments in a digital notebook format. The app helps users record debts, track payments, send reminders, and view reports.

By using the application, you agree to the collection and use of information in accordance with this Privacy Policy.

02

Information We Collect

Account Information

When registering in the application, we may collect the following information:

  • Phone number
  • Name or business name
  • Password

Registration requires verification using a confirmation code.

Customer Information

Users can add customers in order to manage debts. The following information may be stored:

  • Customer name (required)
  • Customer phone number (optional)
  • Debt records
  • Payment history

This information is used only for managing debts within the application.

03

How We Use the Information

The collected information is used to:

  • Create and manage user accounts
  • Store and manage customer debt records
  • Track payments and transactions
  • Send debt reminder messages through the user's SMS application
  • Display statistics and reports
  • Improve the functionality of the application

The application does not sell or share personal data with third parties. User data is stored securely on the application's backend servers and is used only to provide the core functionality of the app.

04

SMS and Phone Features

The application may assist users in contacting customers regarding debts. Features include:

  • Opening the device's default SMS application with a pre-written message
  • Opening the device phone dialer when the user chooses to call a customer
  • Sending SMS reminder messages through an integrated third-party SMS service

For automated SMS delivery, the application may use a third-party SMS provider (Eskiz.uz). When the user sends a reminder message, the customer's phone number and message content may be sent to the Eskiz SMS service to deliver the message. SMS messages may be delivered from a short service number (for example, 4546) depending on the SMS provider configuration.

The application does not automatically send SMS messages or make phone calls without user action.

06

Notifications

The application may schedule local notifications to remind users about debts. These notifications are created within the application and are not sent from external servers.

07

Data Storage and Security

User data is stored securely on backend servers and transmitted using encrypted HTTPS connections. Security measures include:

  • Secure network communication
  • User authentication
  • Protected server storage

We take reasonable measures to protect user data from unauthorized access.

08

Third-Party Services

The application uses third-party services to provide certain features. These services include:

  • Google Firebase Remote Config – used to manage application configuration and version information
  • Eskiz SMS Service (https://eskiz.uz) – used to deliver SMS reminder messages when a user chooses to send them

These services operate under their own privacy policies. The application only sends the minimum required data (such as phone number and message text) necessary to deliver SMS messages when requested by the user.

09

User Data Deletion

Users can delete their account at any time from the application settings. When an account is deleted, the following data will be permanently removed:

  • Account information
  • Customer records
  • Debt history
  • Payment history
10

Third-Party Links

The application may redirect users to external services such as Telegram, Instagram, phone dialer, and SMS application. These services operate under their own privacy policies.

Children's Privacy

This application is intended for business use by shop owners and adults managing customer debts. The app does not knowingly collect personal information from children. The application is rated 3+ on Google Play.
